# Reinstall MacOS

## Install

1. [Download the repo](https://github.com/ArtFlag/reinstall-mac/archive/refs/heads/master.zip) to
   your computer and unpack it.

2. From the root of the folder, run:

   ```console
   sh ./install.sh
   ```
   The script will ask you if you want to set up the shell aliases from the `.aliases` file.

3. Run:

   ```bash
   bash <(curl -sSL https://spotx-official.github.io/run.sh)
   ```

4. Work.
